# BibTeX Field types: edition
Edition of a publication is stored in the field ``edition``. An example for the "second edition":

```tex
@book{Mueller2014The,

  ...

  edition={2.}

  ...

}
```

```tex
@book{Mueller2014The,

  ...

  edition={Second}

  ...

}
```
